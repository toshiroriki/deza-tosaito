# デザートサイト
## サイト概要
デザート専門のecサイト。
絶品な味と写真映え間違いなしの見た目で
味覚、嗅覚など五感全てで堪能いただけます。

### サイトテーマ
多くの人に美味しさを届ける。

### テーマを選んだ理由
世の中には美味しいものでも売れないものがあったり
反対にあまり美味しくないものが売れてたりします。
もっと知ってもらうためにサイトを作りました。

### ターゲットユーザ
インターネット環境がある人が前提で
若い年齢層からご年配の方々。

### 主な利用シーン
誕生日や入学式などの様々なイベント行事。
